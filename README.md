🎨 Tiles Gallery - Assignment 8 (Category-A8-Apple)
This is a modern web application built to showcase a premium tile gallery. Users can explore various tile designs, search for specific styles, and manage their profiles with secure authentication.

🎯 Purpose
The purpose of this project is to build a modern and aesthetic Tiles Gallery platform that allows users to explore premium tile collections. It aims to provide a seamless user experience for searching tiles, viewing detailed descriptions, and ensuring secure user management.

🔗 Live URL
Check out the live project here: https://assignment-eight-project-jbqf.vercel.app/

🚀 Key Features
Secure Authentication: Traditional Email/Password Login and Logout functionality.

Social Login: Quick access via Google Login powered by BetterAuth.

Dynamic Gallery: Users can explore various tile designs fetched from the database.

Tile Details: View high-resolution previews and detailed specifications for each tile.

Search Functionality: Easily find tiles by their names using the search feature.

Profile Management: Logged-in users can view and manage their personal profile data.

🛠️ Technologies Used
Framework: Next.js (App Router)

Styling: Tailwind CSS

Database: MongoDB

Authentication: BetterAuth
